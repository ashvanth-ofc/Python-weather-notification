# Python Weather Notifier 🌦️

A simple Python application that fetches real-time weather data using the Open-Meteo API and displays it as a desktop notification.

## Features
- Search weather by city
- Real-time weather data
- Temperature, wind speed, and wind direction
- Desktop notifications
- Error handling

## Technologies Used
- Python
- Requests
- Plyer
- Open-Meteo API

## How to Run

```bash
pip install requests plyer
python weather.py
```
